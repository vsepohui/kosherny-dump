Чего не хватает в xserver:

1. Единого API как WinAPI (Windows 98), чтобы не приходилось использовать qt/tk.
CreateWindow - и у тебя окно, или кнопка, или что-то еще! (Hdlr - xsocket).

Будет очень здорово писать на Perl:

use AnyEvent::xserver;

my $cv = AnyEvent->condvar;

my $hdlr = AnyEvent::xserver->CreateWindow(width=> 400, height => 100, title => ‘Hello Test’);

$hdlr->on_message( sub{
my ($msg) = @_;
if ($msg->{type} == AnyEvent::xserver::WINDOW_CLOSE) {
$cv->send;
}
});

$cv->recv;
